# Data Availability

This document describes all input data files required to reproduce the analyses in the manuscript,
where each file comes from, and where to place it in the repository.

No data files are committed to this repository. All derived tables are generated by running the
processing pipeline (see [pipeline_overview.md](pipeline_overview.md)).

---

## Public FlyWire / Codex inputs

The following files are available from the FlyWire Codex download page:

> https://codex.flywire.ai/api/download?dataset=fafb

Download the `.csv.gz` files, decompress them, and place them at the paths shown below
(relative to the repository root). Update `config.py` if any filename differs from what is listed.

| File | Local path | Config variable | Description |
|------|-----------|----------------|-------------|
| Princeton synapse detection table | `data/raw/fafb_v783_princeton_synapse_table.csv` | `PRINCETON_SYNAPSE_CSV` | Complete synapse table for FAFB v783. Columns used: `pre_root_id_720575940`, `post_root_id_720575940`, `pre_x/y/z`, `post_x/y/z`, `size`, `neuropil`. After downloading and decompressing, rename the file to match the config variable or update `PRINCETON_SYNAPSE_CSV` in `config.py`. |
| Neurotransmitter-type predictions | `data/raw/neurons.csv` | `NEURONS_CSV` | Per-neuron neurotransmitter predictions. Columns used: `root_id`, `nt_type`. Download `neurons.csv.gz` from the Neurotransmitter Type Predictions section. |
| Cell size statistics | `data/raw/cell_stats.csv` | `CELL_STATS_CSV` | Per-neuron cell size measurements. Expected columns: `root_id`, `length_nm`, `area_nm`, `size_nm`. Download `cell_stats.csv.gz` from the Cell Size Measurements section. |
| Neuron classification hierarchy | `data/raw/classification.csv` | `CLASSIFICATION_CSV` | Per-neuron classification labels. Expected columns: `root_id`, `flow`, `super_class`, `class`, `sub_class`, `hemilineage`, `side`, `nerve`. Download `classification.csv.gz` from the Classification section. Used by `figures/fig3/syntype_x_identity.py`. |
| SWC morphology skeletons | `data/raw/swc/783/` | `SWC_DIR` | One SWC skeleton file per neuron. Download `sk_lod1_783_healed.zip` from the Neuron Skeletons section; extract the archive and place the SWC files in `data/raw/swc/783/`. Verify the internal folder structure of the archive before running the pipeline. |



## Author-provided manuscript inputs

The following files are not available from public databases and must be obtained from the
corresponding authors.

| File | Local path | Config variable | Description |
|------|-----------|----------------|-------------|
| `Supplemental_file1_neuron_annotations.csv` | `data/raw/Supplemental_file1_neuron_annotations.csv` | `NEURON_ANNOTATIONS_CSV` | Combined neuron annotation table. Columns: `root_id`, `super_class`, `top_nt`, `cell_type`. This is the supplementary annotation file from Dorkenwald et al. 2024 (see citation below). Required by `processing/05_build_neuron_metadata_table.py`, `figures/fig2/si_x_primary_types_mirror.py`, and `figures/fig3/si_x_neuron_types.py`. |
| Sensory layer rank feather files | `data/raw/ranks/` | `RANKS_DIR` | Per-neuron sensory layer rank feather files. Columns include `root_id`, `allinputs_layer_mean`, and per-modality layer means (visual, olfactory, mechanosensory, etc.). Required by `processing/05_build_neuron_metadata_table.py`. |
| Neuropil synapse table | `data/raw/Cloud_SQL_fw_mat783_synapses_v3_neuropil.csv` | `NEUROPIL_SYNAPSE_CSV` | Synapse table with neuropil annotations, derived from FlyWire data published in Dorkenwald et al. 2024 (see citation below). Required only by `figures/fig2/si_x_twigs.py` (Fig 2 supplementary panel). |
| Larval synapse table (pre-processed) | `data/larva/output/all_synapses_unprocess_larva_th0.9_SI_filt_issue_solved.ftr` | `LARVA_SYNAPSES_FTR` | Per-synapse larval connectome table with compartment labels, generated by `processing/larva/larva_process.py`. Raw larval data must be obtained from Winding et al. 2023 (see citation below). Required by `figures/fig2/fig2a_si_cdf_adult_larva.py`. |
| Larval SI values (pre-processed) | `data/larva/results/SI_list_larva_th_0.9_linker_SI_filt_issue_solved` | `LARVA_SI_FTR` | Per-neuron larval SI values, generated by `processing/larva/larva_process.py`. Raw larval data must be obtained from Winding et al. 2023 (see citation below). Required by `figures/fig2/fig2a_si_cdf_adult_larva.py`. |

> Dorkenwald S, Matsliah A, Sterling AR, Schlegel P, Yu SC, McKellar CE, et al. Neuronal wiring diagram of an adult brain. *Nature* 634(8032), 124â€“138 (2024). https://doi.org/10.1038/s41586-024-07558-y

The larval `.ftr` output files are generated by running `processing/larva/larva_process.py`.
That script requires three raw input files from the larval connectome published in:

> Winding M, Pedigo BD, Barnes CL, et al. The connectome of an insect brain.
> *Science* 379(6636), eadd9330 (2023). https://doi.org/10.1126/science.add9330

Place these files under `data/larva/raw/` before running `larva_process.py`:

| File | Local path | Config variable | Description |
|------|-----------|----------------|-------------|
| Larval synapse table | `data/larva/raw/synapse_table.csv` | `LARVA_SYNAPSE_TABLE_CSV` | Raw synapse table from the larval connectome. Columns (assigned by script): `rowid`, `pre_skeleton_node_id`, `connector_id`, `pre_x/y/z`, `connector_x/y/z`, `post_skeleton_node_id`, `post_x/y/z`, `pre`, `post`. |
| Neuron filter list | `data/larva/raw/n_list3.csv` | `LARVA_NEURON_LIST_CSV` | CSV listing the larval neuron IDs to include. Column: `neuron`. |
| SWC skeleton files | `data/larva/raw/swc/all_neurons/` | `LARVA_SWC_DIR` | One SWC skeleton file per larval neuron (fixed format with metadata header). |
