# Generated Tables

This document describes all derived tables produced by the processing pipeline.
All files are generated by running `processing/01_*.py` through `07_*.py` in order.
None of these files are committed to the repository — regenerate them from raw inputs.

See [pipeline_overview.md](pipeline_overview.md) for run order and per-script I/O details,
and [data_availability.md](data_availability.md) for how to obtain the required raw inputs.

---

## Primary derived tables (`data/derived/`)

These are the main tables consumed by figure scripts.

### `synapses_783_article_princeton.ftr`

Generated by: `processing/04_build_master_synapse_table.py`

Per-synapse table. This is the central analysis table — almost all figure scripts use it.

| Column | Description |
|--------|-------------|
| `synapse_id` | Unique synapse identifier |
| `pre` | Pre-synaptic neuron root ID |
| `post` | Post-synaptic neuron root ID |
| `comp` | Compartment label: `AA` (axo-axonic), `AD` (axo-dendritic), `DA` (dendro-axonic), `DD` (dendro-dendritic) |
| `SI_pre` | Synaptic Input Index of the pre-synaptic neuron |
| `SI_post` | Synaptic Input Index of the post-synaptic neuron |
| `npil` | Neuropil region |
| `pre_x`, `pre_y`, `pre_z` | Pre-synaptic site coordinates |
| `post_x`, `post_y`, `post_z` | Post-synaptic site coordinates |
| `size` | Synapse size |

---

### `neuron_data_full_article_princeton.ftr`

Generated by: `processing/05_build_neuron_metadata_table.py`

Per-neuron metadata table. Used by virtually all figure scripts.

| Column | Description |
|--------|-------------|
| `root_id` | Neuron root ID (FlyWire segment ID) |
| `super_class` | Neuron super-class (e.g., sensory, motor, central) |
| `primary_type` | Primary neuron type label |
| `nt_type` | Predicted neurotransmitter type |
| `SI` | Synaptic Input Index (fraction of input synapses on dendritic compartment) |
| `cable_length` | Total skeleton cable length |
| `out_synapses_count` | Number of output (pre-synaptic) connections |
| `in_synapses_count` | Number of input (post-synaptic) connections |
| Morphological feature columns | Various per-neuron morphological measurements derived from SWC skeletons |
| Sensory rank columns | `allinputs_layer_mean` and per-modality sensory layer mean values |

---

### `connections_by_syn_type_reciprocal_types_filtered_article_princeton.ftr`

Generated by: `processing/06_build_connection_reciprocity_table.py`

Per-connection table. Used by Figures 3, 4, and 5.

| Column | Description |
|--------|-------------|
| `pre` | Pre-synaptic neuron root ID |
| `post` | Post-synaptic neuron root ID |
| `AA` | Number of axo-axonic synapses in this connection |
| `AD` | Number of axo-dendritic synapses |
| `DA` | Number of dendro-axonic synapses |
| `DD` | Number of dendro-dendritic synapses |
| `sum_syn` | Total synapse count in this connection |
| `reciprocal` | Whether a return connection (post -> pre) exists |
| `same_type` | Whether pre and post neurons share the same primary type |

---

### `neurons_pca_princeton.ftr`

Generated by: `processing/07_pca_morphology.py`

Per-neuron PCA scores from morphological features. Used by Figure 4.

| Column | Description |
|--------|-------------|
| `neuron` | Neuron root ID |
| `PC1` | First principal component score (morphological gradient) |
| `PC2` | Second principal component score |

## Secondary derived tables (`data/derived/`)

These tables are produced as intermediate or auxiliary outputs.

| File | Generated by | Description |
|------|-------------|-------------|
| `SI_updated.ftr` | `05_build_neuron_metadata_table.py` | Per-neuron SI values (multi-pass updated version used for some comparisons) |
| `swc_data.ftr` | `05_build_neuron_metadata_table.py` | Per-neuron SWC summary statistics |
| `reci_prop_princeton.ftr` | `figures/fig5/reciprocal_fraction.py` | Per-neuron reciprocal connection proportion |
| `full_reci_connections_article_filtered_princeton.ftr` | `06_build_connection_reciprocity_table.py` | Full list of reciprocal connection pairs |
| `reciprocity_list_full_article_princeton.ftr` | `06_build_connection_reciprocity_table.py` | Raw reciprocity edge list |
| `PC1_table.csv` | `07_pca_morphology.py` | PC1 scores in CSV format for downstream model scripts |
| `PC1_table_with_predictions.csv` | `figures/fig4/syntype_x_pc1.py` | PC1 table with model predictions appended |
| `synapses_783_article_princeton_raw.ftr` | `04_build_master_synapse_table.py` | Pre-filtered version of the synapse table |
| `neurons_nt_bwf_frac.ftr` | `05_build_neuron_metadata_table.py` | Per-neuron neurotransmitter body-weight fraction (used by Fig 5) |
