# Data directory

This directory holds raw inputs and generated derived tables for the pipeline.
**Two small files are committed to this repository** (`data/raw/Supplemental_file1_neuron_annotations.csv`
and `data/raw/ranks/`). All other data files must be downloaded or generated — see below.

---

## External data sources

Raw and generated data files are **not included** in this repository.

The primary source for FAFB connectome data is the **FlyWire Codex** download page:

> https://codex.flywire.ai/api/download?dataset=fafb

Download the relevant `.csv.gz` files from that page, decompress them, and place them in the paths
described below and in `config.py`.

Project-derived `.ftr` / `.pkl` tables are **generated by running the processing scripts** and are not
committed to this repository. Regenerate them from raw data by running `processing/01_*.py`
through `07_*.py` in order.

Some files (`Supplemental_file1_neuron_annotations.csv`, `ranks/`, and larva data) are not
available from the Codex page and must be obtained from the corresponding authors.
See [docs/data_availability.md](../docs/data_availability.md) for the full data provenance reference.

## Codex files used by this project

| Codex file | Decompressed local name | Local path | Config variable |
|-----------|------------------------|------------|----------------|
| Synapse Table CSV.gz | `fafb_v783_princeton_synapse_table.csv` | `data/raw/` | `PRINCETON_SYNAPSE_CSV` |
| `neurons.csv.gz` | `neurons.csv` | `data/raw/` | `NEURONS_CSV` |
| `cell_stats.csv.gz` | `cell_stats.csv` | `data/raw/` | `CELL_STATS_CSV` |
| `classification.csv.gz` | `classification.csv` | `data/raw/` | `CLASSIFICATION_CSV` |
| `sk_lod1_783_healed.zip` | `*.swc` files (extract to folder) | `data/raw/swc/783/` | `SWC_DIR` |

The following Codex files are **not used** by this pipeline and do not need to be downloaded:
`consolidated_cell_types.csv.gz`, `processed_labels.csv.gz`.

> **Note on synapse table filename:** The Codex download section is labelled "Synapse Table".
> The exact downloaded filename may differ from `fafb_v783_princeton_synapse_table.csv`.
> After downloading and decompressing, rename the file to match, or update `PRINCETON_SYNAPSE_CSV` in `config.py`.

> **Note on skeleton archive:** After downloading `sk_lod1_783_healed.zip`, extract the SWC files
> and verify they land in `data/raw/swc/783/`. Check the internal folder structure of the archive
> before running the pipeline.

## Larva synapse data (`data/larva/`)

Larval data are based on the larval *Drosophila* connectome published in:

> Winding M, Pedigo BD, Barnes CL, et al. The connectome of an insect brain.
> *Science* 379(6636), eadd9330 (2023). https://doi.org/10.1126/science.add9330

No larval data files are committed to this repository. The raw input files from Winding et al. 2023
must be obtained from the corresponding author and placed under `data/larva/raw/`. Once in place,
run `processing/larva/larva_process.py` to generate the `.ftr` files required by figure scripts.

| File | Config variable | Description |
|------|----------------|-------------|
**Raw inputs** (obtain from corresponding author — from Winding et al. 2023):

| File | Config variable | Description |
|------|----------------|-------------|
| `data/larva/raw/synapse_table.csv` | `LARVA_SYNAPSE_TABLE_CSV` | Raw synapse table. Columns: `rowid`, `pre_skeleton_node_id`, `connector_id`, `pre_x/y/z`, `connector_x/y/z`, `post_skeleton_node_id`, `post_x/y/z`, `pre`, `post`. |
| `data/larva/raw/n_list3.csv` | `LARVA_NEURON_LIST_CSV` | Neuron ID filter list. Column: `neuron`. |
| `data/larva/raw/swc/all_neurons/` | `LARVA_SWC_DIR` | One SWC skeleton file per larval neuron. |

**Generated by `processing/larva/larva_process.py`**:

| File | Config variable | Description |
|------|----------------|-------------|
| `data/larva/output/all_synapses_unprocess_larva_th0.9_SI_filt_issue_solved.ftr` | `LARVA_SYNAPSES_FTR` | Per-synapse larval connectome table with compartment labels. Required by `figures/fig2/fig2a_si_cdf_adult_larva.py`. |
| `data/larva/results/SI_list_larva_th_0.9_linker_SI_filt_issue_solved` | `LARVA_SI_FTR` | Per-neuron larval SI values. Required by `figures/fig2/fig2a_si_cdf_adult_larva.py`. |
